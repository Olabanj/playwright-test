# Communication in Dedicated Channels, Not DMs

**Decision:** All PR-related and progress discussions must happen in dedicated project channels — not DMs.

**Why:** Baha and Aleyna's DM-only communication for time tracking left four teammates with zero context. Slahudeen flagged this on 2026-05-14.

**How to apply:** Post PR links immediately when raised. Use the PR review channel workflow. All coordination must be visible to the full team.
